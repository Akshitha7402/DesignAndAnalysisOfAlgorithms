Problem

The prediction of RNA secondary structure using dynamic programming is a classic problem in computational biology, involving the identification of how RNA sequences fold based on their nucleotide interactions. The main goal is to find the most energetically favorable configuration, typically represented by the maximum number of base pairings under certain constraints.

Dynamic Programming

The RNA Folding algorithm builds on the principles established by the Nussinov algorithm, which focuses solely on sequence information to optimize RNA structures by maximizing the number of base pairings. The algorithm uses a recursive approach to solve for optimal substructures incrementally, appending bases one at a time until certain conditions are met. However, it also imposes specific rules to improve the accuracy of its secondary structure predictions:

  • Hairpin Loop Constraints: Hairpin loops in the predicted structure must consist of at least four bases to avoid energetically unfavorable sharp turns.
  • Exclusion of Base Triples: The algorithm ensures that any two base pairs, (i, j) and (i’, j’), must be either identical or have no overlapping indices, effectively eliminating the formation of base triples.
  • Prohibition of Pseudoknots: Structures where one base pair is nested within another but connected in a crossing manner (pseudoknots) are not allowed, maintaining a simpler, planar structure.

These rules help SSFold produce more realistic and stable RNA secondary structures by considering both the quantitative and spatial aspects of molecular folding. The RNA secondary structure prediction problem involves determining the optimal way an RNA sequence can fold to maximize the number of complementary base pairs (A-U and G-C). The sequence folds back on itself, and the structure can be thought of as a set of non-crossing base pairs.

Hello!

Welcome to the documentation for the RNA Secondary Structure Prediction project. This C++ program is designed to solve the RNA secondary structure prediction problem using dynamic programming. The goal is to predict how the RNA molecule will fold back on itself to form pairs between complementary bases, which are typically adenine (A) paired with uracil (U) and cytosine (C) paired with guanine (G).

This documentation covers the introduction to the problem, algorithm and code structure implemented, and the analysis of the code in terms of time and space complexity.

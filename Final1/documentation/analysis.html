Complexity Analysis


Time Complexity: O(n^3), there are O(n^2) subproblems


Space Complexity: O(n^2): storage requirements of the dynamic programming matrix that tracks the optimal number of pairings for subsequences of the RNA.

Why O(n^3)?

Length of the RNA Sequence (n): Complexity increases cubically with RNA length due to n x n DP matrix.

Filling the DP Matrix: Each DP cell's calculation depends on values computed beforehand, maintaining a computation order.

Nested Loops: Three nested loops iterate over the sequence, resulting in O(n^3) complexity due to evaluation of potential pairings.

Recurrence Relation: DP cells are updated by taking the maximum of unpaired bases and optimal pairings.

Overall Time Complexity: Governed by O(n^3) due to nested loops iterating over the DP matrix.

Traceback Function: Polynomial time, usually quicker than DP matrix fill; traces only optimal solutions.

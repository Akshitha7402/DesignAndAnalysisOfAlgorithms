Convex Hull Visualization

Welcome to the Convex Hull Visualization project. This website provides an interactive visualization of two algorithms for finding the convex hull of a set of points: the Jarvis March algorithm and the Kirk Patrick Seidel algorithm.

To get started, navigate to the Algorithms section.

File Structure

The project has the following file structure:

  • index.html: The main HTML file that serves as the homepage of the project.
  • style.css: The CSS file containing styles for the website.
  • main.js: The JavaScript file that handles the interactive visualization on the homepage.
  • algorithms/: A directory containing HTML and JavaScript files for each algorithm.
  • docs/: The directory containing the documentation files.

Code Documentation

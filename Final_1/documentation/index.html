Introduction

Welcome to the documentation for the Convex Hull Visualization project. This project provides an interactive web-based visualization of two algorithms for finding the convex hull of a set of points: the Jarvis March algorithm and the Kirk Patrick Seidel algorithm.

The convex hull of a set of points is the smallest convex polygon that contains all the points in the set. It has various applications in fields such as computational geometry, computer graphics, and pattern recognition.

This documentation covers the implementation details of the two algorithms, their time and space complexities, advantages and disadvantages, and the code structure of the project. It also includes a step-by-step guide on how to use the interactive visualization to explore the working of these algorithms.
